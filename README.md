# Máltækni, verkefni 3

Verkefni sett upp með `poetry`, `pytest` og `black`.

Fyrst þarf að setja upp `poetry` og keyra:

```bash
poetry install
```

Til að formatta kóða:

```bash
poetry run black
```

Til að keyra test:

```bash
poetry run pytest # einu sinni
poetry run ptw    # fylgjast með breytingum á skrám og keyra test
```
